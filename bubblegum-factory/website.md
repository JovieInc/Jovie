# The Bubblegum Factory — Mystery Homepage

The website is minimal. One page. Maximum mystery.

## Design Principles
- All white background (#F8F8F8)
- One hero image: the lobby render
- No navigation menu
- No about section
- No contact form
- Just: Name, tagline, location, logo, one image
