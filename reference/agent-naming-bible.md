# Agent Naming Bible

**A reference for naming all future AI agents in the ecosystem.**

## Rules

1. **All female names** — never male
2. **Futuristic / sci-fi inspired** — cool, fun, sexy, flirty vibes
3. **Clean and easy to spell** — easy to Google, no weird spelling
4. **TV character potential** — could work as a fictional character name
5. **Rare but not weird** — Summer is the gold standard. Recognizable feel but not common
6. **NOT celebrity/common names** — no Sydney, no Sydney Sweeney adjacent, no Olivia
7. **First names only** — last names come later
8. **Fantasy/sci-fi adjacent is good** — think space, tech, future, mythology

## Naming Patterns & Themes

### Celestial / Space
Names that evoke stars, planets, moons, cosmology. Feels expansive and futuristic.

### Retro-Futuristic
Names that feel like they're from the future but still grounded. Sci-fi but not fantasy.

### Nature-Tech Hybrids
Names that blend organic/natural elements with a tech edge. Like "Summer" — natural but cool.

### Mythological (Light Touch)
Names inspired by mythology but not literal goddess names. Subtle, not costume-y.

### Phonetic Patterns That Work
- Strong consonants + soft vowels (St-e-ll-a, A-ri-a)
- Two syllables is the sweet spot, three works too
- Names that end in "a" or "ia" have a natural flow
- Avoid names that rhyme with each other within the ecosystem
