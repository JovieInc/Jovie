# Printing Press

**Creator**: Matt Van Horn (@mvanhorn)
**Stars**: 4k+
**Type**: Open-source CLI toolkit

## What It Does
Fleet of CLI tools for real-world tasks:
- Tesla preheat
- Instacart orders
- Flight booking
- Sports watching
- Granola meeting transcription access

Works with **Agent Cookie** to let agents use your real browser session.

- Directly applicable to Jovie booking flows (venue outreach, travel)
- Agent Cookie concept is similar to how we'd want agents to interact with authenticated services
