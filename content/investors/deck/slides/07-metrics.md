# What We Measure

Early, measurable KPIs with a simple value ladder.

- **Release velocity:** releases shipped per artist over time
- **Capture rate:** profile views → email/SMS signup
- **Activation rate (24h):** captured → meaningful action (listen/follow/save/tour intent)
- **Release-loop activation:** release created → notification sent → pitch generated → fan action
- **Value events per fan/week:** streaming clicks + follow/save + ticket/merch intent
- Weighted event scoring before full purchase attribution is available
