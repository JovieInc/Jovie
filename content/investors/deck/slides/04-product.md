# The Product

AI flywheel for fan value extraction — runs on every profile view.

1. **Identify** the fan (known user, captured email/SMS, or anonymous)
2. **Read** fan state (subscription, platform preference, geo, recency)
3. **Decide** next best action (1 primary CTA + 1-2 secondary)
4. **Measure** impressions → clicks → conversions → downstream value
5. **Learn** experiments and segmentation improve decisions over time
