# Features

Jovie includes a comprehensive set of tools built specifically for musicians.

- [Profile](/docs/features/profile) -- Your public artist page with bio, photos, and links.
- [Releases and Smart Links](/docs/features/releases) -- Manage your releases and generate smart links that route fans to the right platform.
- [Audience](/docs/features/audience) -- Understand and segment your fan base.
- [Chat and AI](/docs/features/chat-ai) -- Communicate with fans using AI-assisted messaging.
- [Tips](/docs/features/tips) -- Accept tips and support from your fans directly.
- [Analytics](/docs/features/analytics) -- Track profile views, link clicks, and audience growth.
