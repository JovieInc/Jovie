# Releases and Smart Links

Manage your music releases and create smart links that route fans to their
preferred streaming platform.

## How it works

1. **Add a release** -- Enter your release details or let Jovie auto-detect them from connected DSPs.
2. **Smart link generation** -- Jovie creates a single link that detects the listener's preferred platform and redirects them accordingly.
3. **Share everywhere** -- Use your smart link in social bios, stories, emails, and more.

## Supported platforms

Jovie supports all major streaming and download platforms, including Spotify,
Apple Music, YouTube Music, Amazon Music, Tidal, Deezer, SoundCloud, and more.

## Pre-save campaigns

Set up pre-save campaigns so fans can save your upcoming release before it drops.
Jovie handles the OAuth flow and saves automatically on release day.
