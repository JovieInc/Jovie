# Ad Pixel Tracking

Install tracking pixels on your Jovie pages to build retargeting audiences and measure ad campaign performance across Facebook, Google, and TikTok.

## Supported platforms

| Platform | Pixel type | What it enables |
|---|---|---|
| Facebook / Meta | Facebook Pixel + CAPI | Retarget Jovie visitors on Facebook and Instagram ads |
| Google | Google Analytics | Track Jovie traffic in Google Analytics, build audiences for Google Ads |
| TikTok | TikTok Pixel | Retarget Jovie visitors on TikTok ads |

## How it works

1. **Add your pixel IDs** -- Go to Settings > Ad Pixels and enter your pixel IDs for each platform.
2. **Automatic event forwarding** -- Jovie fires pixel events on page views and link clicks across your profile and smart link pages.
3. **Build audiences** -- Use the collected data in your ad platform to create retargeting audiences.
4. **Measure performance** -- See which ad campaigns drive traffic to your Jovie profile and which fans convert.

## Key capabilities

- **Server-side event forwarding** -- Facebook CAPI support for reliable tracking even with ad blockers.
- **Consent management** -- Pixel events respect visitor consent preferences.
- **UTM parameter capture** -- Incoming UTM parameters are tracked and available in your analytics.
- **Session tracking** -- Full session data including IP, user agent, and referrer is forwarded for ad optimization.

## Plan availability

Ad pixel tracking is available on Pro and Growth plans.

## Getting started

1. Upgrade to Pro if needed.
2. Go to **Settings > Ad Pixels**.
3. Enter your Facebook Pixel ID, Google Analytics ID, or TikTok Pixel ID.
4. Pixels start firing immediately on all your Jovie pages.

See the [Set Up Ad Pixels guide](/docs/self-serve-guide/set-up-ad-pixels) for detailed instructions.
