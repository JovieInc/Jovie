## Content Pipeline

1. Generate set image (use `bubblegum-factory/prompts`)
2. Generate video (Runway/Kling/Sora)
3. Generate voice (ElevenLabs Tim voice clone)
4. Composite in CapCut/Premiere
5. Add captions + text in post (NEVER generated)
6. Export 9:16 vertical → ship to Shorts/Reels/TikTok
