## Evidence

* Screenshot/frame: 90-second founder demo on /demovideo
* Route: /demovideo
* Observed behavior: The demo shows Tim clicking Approve drop and Jovie completing campaign setup steps.
* Expected behavior: Approving a recommendation should orchestrate campaign creation, smart link updates, notification drafts, audience selection, task creation, scheduling, and monitoring setup.

Shown cinematically in PR [JovieInc/Jovie#8671](https://linear.app/jovie/review/fixdemo-stabilize-public-demo-video-embed-82094abf613c); production implementation needed.

## Impact

This is the demo's core execution proof: one approval should do the work across Jovie-owned surfaces.

## Likely root cause

There is no production orchestration layer that turns approved recommendations into a durable multi-step campaign workflow.

## Acceptance criteria

* Approval starts an idempotent workflow with visible step status.\\n- Each step records success/failure and retry state.\\n- Partial failures do not leave hidden inconsistent state.\\n- Artists can review/edit generated artifacts before launch when needed.

## Test plan

* Workflow unit tests for idempotency and retries.\\n- Integration test for approval to scheduled campaign.\\n- Manual check of step status UI.\\n- Failure injection test for one failed downstream step.

## Suggested fix

Use the existing workflow/job infrastructure to create a campaign approval orchestrator with typed step outputs and resumable execution state.
