# JOV-1938: Add design surface registry + proposal scoring

- **Status:** Backlog
- **Priority:** 🟡 Medium
- **Assignee:** Unassigned
- **Team:** JOV
- **Project:** Design Lab
- **Labels:** agentos
- **URL:** https://linear.app/jovie/issue/JOV-1938/add-design-surface-registry-proposal-scoring
- **Updated:** 2026-05-31T02:17:40.968Z

Define the registry of design surfaces (logo, profile page, email header, sidebar, etc.) with: surface ID, display name, description, weight (used for prioritizing proposal generation), and last-reviewed timestamp. Scoring algorithm uses weight + recency + taste memory rejection count to rank which surfaces get proposals in each daily run. Registry lives in `agentos/agents/design-lab/surfaces.json`.
