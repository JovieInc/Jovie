# JOV-2650: Build merch generation pipeline for mockups and sellable variants

- **Status:** Todo
- **Priority:** 🟡 Medium
- **Assignee:** Tim White
- **Team:** JOV
- **Project:** —
- **Labels:** area:core-flows, mvp, type:feature
- **URL:** https://linear.app/jovie/issue/JOV-2650/build-merch-generation-pipeline-for-mockups-and-sellable-variants
- **Updated:** 2026-05-29T01:46:41.319Z

Implement the merch generation pipeline so it can take generated design assets and produce sellable merch outputs.

The flow should support:

* Design image generation via an interchangeable image generation provider
* Applying designs to merch mockups, either through Printful mockup generation or an internal templating flow
* Outputting sellable variants, mockup images, sale price, and profit

Keep the workflow simple and avoid exposing advanced pricing knobs in the UI.
