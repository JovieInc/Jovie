# JOV-2002: feat(agentos): wire model router into AgentBrief dispatch

- **Status:** Backlog
- **Priority:** 🟡 Medium
- **Assignee:** Unassigned
- **Team:** JOV
- **Project:** —
- **Labels:** codex
- **URL:** https://linear.app/jovie/issue/JOV-2002/featagentos-wire-model-router-into-agentbrief-dispatch
- **Updated:** 2026-05-31T04:59:06.493Z

Wire the model router to read `proposedModelRoute` from AgentBrief and select the correct model route at dispatch time.

## Flow

1. `generateBrief(issueId)` → returns `AgentBrief` with optional `proposedModelRoute`
2. Router reads `proposedModelRoute` → looks up `ModelRoutePolicy` from registry
3. If no `proposedModelRoute` → fallback to `AGENT_RUN_MODEL_ROUTES.deterministic`
4. Dispatch uses the resolved model route

## Blocked by

* [JOV-1933](https://linear.app/jovie/issue/JOV-1933/build-agent-readable-issue-brief-generator) (AgentBrief generator must exist)
* Phase 2 AgentRunArtifact extension (must add `proposedModelRoute` to AgentBrief)

## Acceptance criteria

* Router function: `resolveModelRoute(brief: AgentBrief): ModelRoutePolicy`
* Dispatch path updated to call router before execution
* Typecheck passes
