   * `https://stg.jov.ie/api/connectors/google/callback`
   * `https://jov.ie/api/connectors/google/callback`
   * `http://localhost:3000/api/connectors/google/callback` (dev)
2. **Doppler secrets** (`jovie-web` project, `stg` + `prd` configs, all `type: sensitive`):
   * `GOOGLE_OAUTH_CLIENT_ID`
   * `GOOGLE_OAUTH_CLIENT_SECRET`
   * `GOOGLE_OAUTH_REDIRECT_URI_BASE` = `https://stg.jov.ie/api/connectors/google` (or prd equivalent)
3. **Enable** `AI_CONNECTORS_BETA` **Statsig gate** for 5–10 design-partner DJ user IDs. (Default off; gate is wired in C-PR-3.)
4. **WAIT until **[**JOV-2235**](<https://linear.app/jovie/issue/JOV-2235>)** (cost-anomaly auto-rollback hook) ships** before flipping the flag in prd. Otherwise an extractor regression risks runaway Gateway spend. [JOV-2235](https://linear.app/jovie/issue/JOV-2235/l-11-cost-anomaly-auto-rollback-hook-for-ai-connectors-beta-flag) is in backlog and can be dispatched once production cycle settles.
