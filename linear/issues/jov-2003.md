# JOV-2003: feat(agentos): free-model broker — safe task types only

- **Updated:** 2026-05-31T04:59:05.159Z

Overflow broker that routes safe task types to free/cheap models before spending on paid tiers.

## Safe task types (free-model eligible)

* summarize
* classify
* rank
* draft (non-critical)

## Unsafe task types (always paid tier)

* implement (code changes)
* review (security/billing/auth)
* migrate (DB)
* deploy

## Broker behavior

1. Check task type against allowlist
2. If safe → route to `kilo-free` or `openrouter-cheap`
3. Record attempt in AgentRunArtifact: `attemptNumber`, `modelProvider`, `modelName`
4. Record outcome: pass/fail, human accept/reject (stored in `ops_events` if rejected)
5. If free model fails or human rejects → escalate to paid tier (not auto-retry)

## Acceptance criteria

* `safeTaskTypesForFreeModel` set exported from `apps/web/lib/agent-os/`
* Broker function: `routeWithFreeModelBroker(brief, taskType): Promise<AgentRunArtifact>`
* Outcome recorded in artifact with `attemptNumber` and `autonomyLevel`
* Typecheck passes
