## Problem

On the `/admin/touring` screen, the "Copy link" button uses a different shape (border-radius, padding, or icon-only vs labelled) than the sibling action buttons in the same row or toolbar.

This inconsistency suggests the button was added ad-hoc without checking the surrounding button pattern.

## Expected

* "Copy link" button uses the same `variant` and `size` as sibling action buttons in its row
* If sibling buttons are icon+label, "Copy link" should also be icon+label
* If sibling buttons are icon-only, "Copy link" should be icon-only with a tooltip
