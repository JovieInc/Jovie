# JOV-2102: design(admin/settings): normalize heading typography — replace all-caps and oversized headings

## Problem

The `/admin/settings` screen uses inconsistent heading styles:

* Some section headings are all-caps (banned by the design system — use Title Case)
* Some headings are oversized (`text-xl` or `text-2xl`) where `text-sm font-medium` or `text-base font-semibold` is appropriate for settings sections
* The typographic hierarchy doesn't match the rest of the admin

## Required

Audit all headings in `/admin/settings` and:

1. Convert any all-caps headings to Title Case
2. Reduce oversized section headings to match the admin design system typographic scale
3. Ensure heading levels (`h2`, `h3`, etc.) match semantic hierarchy, not visual size

## Design reference

* Section headings in admin settings: `text-sm font-medium text-muted-foreground` (label style)
* Group/page headings: `text-base font-semibold` or the canonical `CardTitle` component
* No all-caps anywhere — see `.claude/rules/ui.md` → Text Casing Rules
