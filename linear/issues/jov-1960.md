## Why pair-review

* It's an architectural foundation — future AgentOS work will build on `withWorkflow()` shape and the `AGENT_OS_RUNTIME_*` gate semantics. Refactor cost compounds if we lock in a shape we'd change later.
* This is the [JOV-1922](https://linear.app/jovie/issue/JOV-1922/agentos-architecture-decision-record-for-vercel-workflow-control-plane) (workflow ADR, already Done) implementation step. Worth confirming the shape matches the ADR's accepted decisions before merge.
* Per release rules, anything that reshapes runtime composition (Next config) deserves human eyes for blast-radius assessment.
