## Problem

The `/admin/investors` page has accumulated placeholder chrome — empty sections, stub cards, and headings that don't have associated data or actions. It feels unfinished and adds cognitive load without value.

## Required

1. **Remove empty or placeholder sections** — if a section has no data and no clear timeline for data, remove it rather than showing an empty state
2. **Simplify the layout** — if the page can be expressed as a single focused view rather than multiple stacked cards, prefer that
3. **Audit each card/section for purpose** — each element on the page must have a clear question it answers for an admin reviewing investor relations

## Acceptance criteria

* No empty placeholder sections or cards
* Every visible element has real data or a clear empty state with a specific action
* Page loads and renders without layout-shift or skeleton flash
* Design matches admin design system
