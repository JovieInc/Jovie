## Context

PR #8159 is a dependabot-driven Electron major-version bump (34.5.8 → 39.8.5) for `apps/desktop`.

This was deferred from the 2026-05-08 autonomous PR landing rollout per autoplan dual-voice review. Both Codex and Claude opus subagent independently flagged this as not belonging in a product release wave.

## Why deferred

* Major-version dep bump on desktop while the desktop app isn't on the active release path
* Bundling it into a product release wave (web app PRs landing same day) makes incident attribution harder
* Dependabot will reopen against fresh main if it's left to drift

## When to reconsider

* Next active desktop release cycle, OR
* Sooner if a security advisory emerges for Electron 34.x, OR
* Sooner if the desktop app starts shipping regularly

## Action

- [ ] When picking this up: rebase against fresh main (lockfile likely needs regen — see prior agent fix in same branch)
- [ ] Run desktop-specific QA (the standard web /qa won't cover Electron-specific regressions)
- [ ] Review Electron 39 breaking changes from changelog
- [ ] Land in isolation (not bundled with web product PRs)
