# JOV-2093: design(admin/tasks): stale task cards — overdue badge clips and layout needs refresh

- **Status:** Backlog

- **Assignee:** Unassigned
- **Team:** JOV
- **Project:** —

## Problem

The admin Tasks page has two related design issues:

1. **Overdue badge clips** — the "Overdue" badge on task cards is clipped or partially hidden behind the card boundary. The badge needs sufficient padding and the card layout must accommodate it without overflow.
2. **Stale task card design** — the overall task card design has not been updated to match the current admin design system. Cards feel visually inconsistent with the rest of the admin UI (spacing, typography, surface treatment).

## Expected

* Overdue badge renders fully within the card boundary with correct padding
* Task cards use current design tokens (surface, border, text sizing, spacing)
* Card layout matches the admin design system at all viewport widths

## Scope

* `/admin/tasks` route
* Task card components
* Badge rendering in card context
