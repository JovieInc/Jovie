# JOV-2007: feat(agentos): Brain panel in /app/admin/ops HUD

- **Status:** Backlog
- **Priority:** 🟡 Medium
- **Assignee:** Unassigned
- **Team:** JOV
- **Project:** —
- **Labels:** codex
- **URL:** https://linear.app/jovie/issue/JOV-2007/featagentos-brain-panel-in-appadminops-hud
- **Updated:** 2026-05-31T04:59:00.673Z

Show new FounderDecisionMemory rule candidates in the /app/admin/ops HUD. Tim approves, rejects, or annotates proposed rules.

## Scope

* Brain panel component in `/app/admin/ops` page
* Fetches pending proposals from `GET /api/admin/agent-os/memory/proposals`
* `PATCH` approve/reject actions wired to `PATCH /api/admin/agent-os/memory/proposals/:id`
* Approved rules auto-append to the relevant `agentos/memory/*.md` file

## Blocked by

* [JOV-2005](https://linear.app/jovie/issue/JOV-2005/featagentos-add-founderdecisionmemory-type-apiadminagent) (FounderDecisionMemory type + proposals endpoint must exist first)

## Acceptance criteria

* Panel renders a list of pending rule proposals
* Each row shows: rule text, reasoning, proposed-at timestamp
* Approve → status flips to approved, rule appended to memory file
* Reject → status flips to rejected, row dismissed
* Empty state: "No pending proposals"
