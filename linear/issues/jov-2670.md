# JOV-2670: UX: Hide chat suggestions for unconfigured album-art provider

- **Status:** Todo

- **Assignee:** Unassigned

- **Project:** —
- **Labels:** grok
- **URL:** https://linear.app/jovie/issue/JOV-2670/ux-hide-chat-suggestions-for-unconfigured-album-art-provider
- **Updated:** 2026-05-31T07:41:48.000Z

## Problem

Chat shows suggestions for album art generation even when the user hasn't configured an album art provider. This is confusing and makes the feature look broken.

## Scope

* Check if album-art provider is configured before showing suggestions
* Hide suggestion chips when no provider is set
* Show a subtle CTA to configure provider instead

## Acceptance

* No album art suggestions when provider not configured
* Clear path to configure provider
* Suggestions appear immediately after configuration
