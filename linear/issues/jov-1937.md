Persistent store of accepted and rejected design directions at `agentos/memory/design-taste.md`. Each entry records: surface, direction, decision (accepted/rejected), notes, and timestamp. This file is injected into every Design Lab agent run as system context so the loop cannot regenerate rejected directions. Scoped narrower than [JOV-1888](https://linear.app/jovie/issue/JOV-1888/agent-os-gbrain-company-state-sync) (GBrain sync) — this is local execution memory only, not GBrain-synced.
