# JOV-2551: Make chat-first golden path error-free and accessible

- **Status:** Todo
- **Priority:** 🟠 High
- **Assignee:** Tim White
- **Team:** JOV
- **Project:** —
- **Labels:** codex

- **Updated:** 2026-05-31T04:56:45.965Z

Implement the chat-first golden path reliability plan:

* Persist anonymous `/api/chat` onboarding transcripts durably by signed onboarding session.
* Claim anonymous transcripts after signup.
* Adopt claimed conversations onto the creator profile in `/api/onboarding/welcome-chat` so the final `/app/chat` route can load the transcript.
* Preserve Turnstile, rate limits, relative return validation, and checkout handoff behavior.
* Add focused unit and Playwright coverage for zero browser errors, accessibility-sensitive states, and layout stability.
* Verify typecheck, focused tests, Biome, and browser screenshots.
