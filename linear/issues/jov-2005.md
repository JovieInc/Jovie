REST endpoint for the HUD brain panel. Stores proposed policy rules from agents for Tim to approve/reject.

## Interface

```ts
// POST /api/admin/agent-os/memory/proposals
// body: { rule: string; reasoning: string; sourceContext?: string }
// returns: { id: string; rule: string; reasoning: string; status: 'pending'; createdAt: string }

// GET /api/admin/agent-os/memory/proposals
// returns: ProposalSummary[]

// PATCH /api/admin/agent-os/memory/proposals/:id
// body: { status: 'approved' | 'rejected'; note?: string }
// on approved: append rule to relevant agentos/memory/*.md file
```

## Storage

* Proposals stored in DB table `founder_decision_proposals` (new migration)
* On approve: append to appropriate `agentos/memory/*.md` file via filesystem write + git commit
* On reject: mark dismissed, keep in DB for audit trail

## Auth

* All endpoints require `isAdmin` entitlement check

## Acceptance criteria

* Zod types in `apps/web/lib/agent-os/founder-decision-memory.ts`
* API routes at `apps/web/app/api/admin/agent-os/memory/proposals/`
* DB migration for `founder_decision_proposals` table
* Approved rules appended to memory files (filesystem write)
* Typecheck + Biome pass
