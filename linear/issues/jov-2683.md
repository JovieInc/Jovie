## Goal

Create a distinctive, high-quality voice for Jovie — the virtual co-host character in the Bubblegum Factory universe.

## Character Brief

Jovie is:

* A virtual AI platform representation / co-host
* Sharp, opinionated, artistically fluent
* Sometimes right, sometimes wrong, always interesting
* Female-presenting (likely direction)
* Sounds creative, slightly irreverent, confident
* Age feel: mid-20s, LA-native energy

## Approach

Since Jovie is fictional, we cannot clone from real audio. Options:

### Option A: Curate an existing ElevenLabs voice

1. Browse ElevenLabs voice library for voices matching the character
2. Test top 3-5 candidates with demo scripts
3. Fine-tune voice settings (stability, similarity, style)
4. Save the voice_id

### Option B: Voice design via ElevenLabs Speech Synthesis

1. Use a base voice close to target
2. Adjust voice_settings to match character
3. Generate demos and iterate

## Demo Scripts to Generate (for each candidate voice)

1. Bubblegum Factory intro: "Welcome to the Bubblegum Factory. I'm Jovie. Let's talk about what's actually happening in the music industry."
2. Artist advice: "Look, I'm going to be real with you. Your mix is not the problem. Your distribution strategy is. Let me show you what I mean."
3. Banter with Tim: "Tim, that is the most founder thing you have ever said. But you are not wrong."
4. Industry take: "The music industry does not have a talent problem. It has a tools problem. And that is exactly why we are here."

## Deliverables

* Selected voice_id saved to \~/youtube-training-data/jovie_voice.json
* Demo audio files (at least 4, one per script above)
* Voice settings documented (model, stability, similarity, style, speaker_boost)
* Playback so Tim can hear and iterate

## Env

* ELEVENLABS_API_KEY in 1Password and Doppler (jovie-web/prd)
* Use `doppler run --project jovie-web --config prd -- python3 ...` to inject key
