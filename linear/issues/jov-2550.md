# JOV-2550: Re-evaluate iOS public launch

- **Status:** Todo
- **Priority:** 🟡 Medium
- **Assignee:** Tim White
- **Team:** JOV
- **Project:** —
- **Labels:** codex
- **URL:** https://linear.app/jovie/issue/JOV-2550/re-evaluate-ios-public-launch
- **Updated:** 2026-05-31T04:56:48.211Z

## Decision

Ship now: internal TestFlight Track A to admins/internal testers only. Public iOS marketing stays disabled.

## Re-evaluate when

* 7 days with no P0/P1 iOS issues.
* Crash-free sessions >= 99.5%.
* Install-to-auth completion >= 85%.
* `/api/mobile/v1/me` success >= 99%.
* Profile QR render p95 <= 250ms cached.
* TestFlight processing/installability check is green.
* Screenshot QA has zero known layout-shift defects.

## Then

Enable Track B native text chat to the same alpha cohort when:

* Chat send success >= 97%.
* p95 cached conversation open <= 500ms.
* p95 network conversation load <= 3s.
* Duplicate-turn replay tests pass.
* Offline retry survives app restart.

Only after these gates should the public `/download` iOS CTA and broader alpha cohort be enabled.
