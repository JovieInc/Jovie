## Problem

Product screenshots mix real artists with fake/generic data. "Don't Look Down" is a test song. Real releases are missing collaborator credits (e.g. Tom Fall). Screenshots invent song names and artist credits.

## Acceptance Criteria

- [ ] Canonical title, artists, album art, duration, and Spotify IDs pulled from Spotify for any real artist fixture
- [ ] Fake status/date/stage allowed only in explicitly named demo fields
- [ ] Fake titles/artists/artwork banned in real-artist fixtures
- [ ] "Don't Look Down" and other placeholder titles removed from fixtures
- [ ] Tom Fall and all collaborators correctly credited
- [ ] Fixture records created from Spotify import and snapshotted

## Invariant Test

* If release has Spotify ID → title/artists/artwork must come from Spotify-derived canonical data
* Test fails if release title contains known placeholder names
