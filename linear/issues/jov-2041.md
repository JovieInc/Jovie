## What Tim needs to do

The `gbi` inbox CLI ships with an iOS voice-capture path (Shortcut → Working Copy → git push → cron drain → gbrain). The macOS side is fully automated and live (drain timer at `co.timwhite.gbi.drain`, every 5 min). The iOS side requires manual setup.

## Steps

### 1\. Install Working Copy on iPhone

* Download from the App Store (free; $20 lifetime unlock for write+push, which you need)
* Open the app, settings → SSH keys → generate or import key
* Add the public key to GitHub ([https://github.com/settings/keys](<https://github.com/settings/keys>))

### 2\. Clone the inbox-drops repo in Working Copy

* Working Copy: + → Clone Repository → SSH
* Paste: `git@github.com:itstimwhite/inbox-drops.git`
* Confirm pull works

### 3\. Build the iOS Shortcut

Open Shortcuts app → + → name it "Inbox capture" → add these actions in order:

| \# | Action | Settings |
| -- | -- | -- |
| 1 | Dictate Text | Language: English (US). Stop Listening: After Pause |
| 2 | Format Date | Date: Current Date · Format: Custom · `yyyyMMdd-HHmm` |
| 3 | Text | `inbox-[Formatted Date].md` (insert variable from #2) |
| 4 | Working Copy: Write Repository File | Repo: inbox-drops · Path: `pending/[Text from #3]` · Content: \[Dictated Text from #1\] |
| 5 | Working Copy: Commit & Push | Repo: inbox-drops · Message: `ios: capture [Formatted Date]` · Push after commit: ON |
| 6 | Show Notification (optional) | Title: "Captured" · Body: \[Dictated Text\] |

### 4\. Add to Siri

* Long-press the shortcut → Add to Siri → record phrase: **"capture thought"**
* Test: `Hey Siri, capture thought` → speak → push

### 5\. Verify end-to-end

* After saying "Hey Siri, capture thought", wait up to 5 min
* Run `gbi list` on laptop → should show your captured thought as a pending inbox page
* Run `gbi triage` to classify it

## Why this is human-only

* Shortcuts only live on iPhones; no API to create them programmatically
* Working Copy SSH key generation requires Face ID + a phone screen
* Siri recording requires the user's voice
* App Store purchase requires Tim's Apple ID
