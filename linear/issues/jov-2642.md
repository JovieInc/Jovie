# JOV-2642: Promote Library filters into top-level chips

- **Status:** Backlog
- **Priority:** 🟠 High
- **Assignee:** Tim White
- **Team:** JOV
- **Project:** —
- **Labels:** —
- **URL:** https://linear.app/jovie/issue/JOV-2642/promote-library-filters-into-top-level-chips
- **Updated:** 2026-05-31T02:15:14.561Z

Restructure Library filtering so key content types become top-level chips or tabs above the main content area instead of a long nested filter stack.

Primary chips should cover content types like All, Releases, Merch, Images, Videos, and Audio. Secondary chips should cover common states such as Draft, Released, Missing artwork, and Missing providers.

This should make the Library controls feel like content filters rather than a left sidebar embedded in the page.
