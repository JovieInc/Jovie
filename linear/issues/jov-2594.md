Replace the dedicated playlist-pitching UI with a centralized chat skill/tool flow.

Scope:

* Remove the release sidebar pitch tab and target-playlists pitching UI.
* Add a Generate pitch action to release and pitch-related task action menus.
* Route pitch generation through Jovie chat and the existing tool/artifact system.
* Infer destination from task context where possible and ask the user where they want to pitch when unclear.
* Keep V1 draft-only so output can be copied from chat; Gmail draft/send remains connector-gated and not implemented here.
