# JOV-2663: UX: Default Library to list view

- **Status:** Todo
- **Priority:** 🟠 High
- **Assignee:** Unassigned
- **Team:** JOV
- **Project:** —
- **Labels:** grok
- **URL:** https://linear.app/jovie/issue/JOV-2663/ux-default-library-to-list-view
- **Updated:** 2026-05-31T07:27:24.000Z

## Problem

The Library defaults to grid view which wastes space with small thumbnails. List view is more scannable and better for power users.

## Scope

* Change default Library view from grid to list
* Persist user preference in localStorage
* Ensure list view has proper row spacing
* Add toggle in header to switch views

## Acceptance

* New users see list view by default
* Toggle works and persists
* No regression in grid view
