Audit which Lighthouse + e2e lanes mutate the shared Neon ephemeral branch. Both review voices read the topology differently:

* Claude subagent: `ci.yml:373` `neon-db` job creates one branch, all downstream lanes `needs: [neon-db]` and share via artifact
