Parent epic for the Phase 7 department agent system. Eight specialized agent teams, each owning a KPI domain and receiving dispatch via AgentBrief.

## Teams

1. Reliability dept
2. Product/Activation dept
3. Coding dept (wraps existing autopilot/orchestrate skills)
4. Design/Taste dept
5. Growth/Revenue dept
6. CFO/Cost dept
7. Founder Brain dept
8. Fundraising/Investor dept

## Prerequisites

* [JOV-1933](https://linear.app/jovie/issue/JOV-1933/build-agent-readable-issue-brief-generator) AgentBrief generator must be live
* Phase 2 AgentRunArtifact + proposedModelRoute extension must be live
* Phase 3 model router must be live
* At least one live dept agent (Coding dept) proving the pattern

## Acceptance criteria

* Each dept agent has a defined brief schema, KPI owner, and dispatch trigger
* Agents record runs as AgentRunArtifacts
* Coding dept is the pilot; remaining 7 follow the same pattern
