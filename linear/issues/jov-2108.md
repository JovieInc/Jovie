# JOV-2108: design(admin): global design-system conformity pass — buttons, cards, tables, badges, inputs

- **Status:** Backlog
- **Priority:** 🔵 Low
- **Assignee:** Unassigned
- **Team:** JOV
- **Project:** —
- **Labels:** codex
- **URL:** https://linear.app/jovie/issue/JOV-2108/designadmin-global-design-system-conformity-pass-buttons-cards-tables
- **Updated:** 2026-05-31T05:03:32.583Z

## Summary

A full sweep of all admin screens to replace off-system UI primitives with canonical design system components. This is the capstone cleanup task after targeted per-screen fixes land.

## Scope

Audit and replace non-conforming instances of:

* **Buttons** — wrong variant, wrong size, wrong shape, custom styles instead of `<Button>` variants
* **Cards** — custom `div`-based cards instead of `<Card>` / `<DrawerSurfaceCard>`, wrong elevation, missing borders/shadows
* **Tables** — custom table markup instead of the shared table subsystem (`organisms/table/`)
* **Badges** — custom status chips instead of `<Badge>` with correct variant
* **Filters** — custom filter UIs instead of canonical filter components
* **Toggles / Switches** — custom toggle implementations instead of `<Switch>`
* **Textareas / Inputs** — custom-styled form elements instead of design system form atoms

## Screens to audit

* `/admin/overview`
* `/admin/ops`
* `/admin/people`
* `/admin/growth`
* `/admin/investors`
* `/admin/settings`
* `/admin/tasks`
* `/admin/touring`

## Approach

1. grep for non-canonical patterns per component type across `components/features/admin/`
2. Replace with canonical imports from `@jovie/ui` or `apps/web/components/atoms|molecules|organisms/`
3. Verify no visual regressions on each screen

## Dependencies

Should be done after (or in parallel with) the targeted per-screen fixes:

* [JOV-2097](https://linear.app/jovie/issue/JOV-2097/designadminops-add-control-panel-card-for-operational-toggles) (ops screen redesign)
* [JOV-2098](https://linear.app/jovie/issue/JOV-2098/designadmin-consolidate-overlapping-ia-across-overview-ops-growth) (admin IA consolidation)
* [JOV-2099](https://linear.app/jovie/issue/JOV-2099/designadminpeople-migrate-people-tables-to-shared-admin-table-system) (People table refactor)
