Postgres-backed `ops_events` table (Drizzle migration, additive only — never edit existing migrations).

## Schema

```sql
CREATE TABLE ops_events (
  id uuid PRIMARY KEY DEFAULT gen_random_uuid(),
  severity text NOT NULL CHECK (severity IN ('critical','high','medium','low')),
  impact text NOT NULL CHECK (impact IN ('blocks_revenue','degrades_ux','infra_risk','info')),
  source text NOT NULL CHECK (source IN ('sentry','ci','linear','deploy','manual')),
  title text NOT NULL,
  body text,
  linked_issue_id text,
  linked_pr_number int,
  linked_run_id uuid REFERENCES agent_run_artifacts(id),
  resolved_at timestamptz,
  created_at timestamptz NOT NULL DEFAULT now()
);
CREATE INDEX ops_events_severity_impact ON ops_events (severity, impact);
CREATE INDEX ops_events_source_created ON ops_events (source, created_at DESC);
```
