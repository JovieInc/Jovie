# JOV-1932: Build `/roadmap` command parser for Linear ops

- **Status:** Backlog
- **Priority:** 🟠 High
- **Assignee:** Unassigned
- **Team:** JOV
- **Project:** Roadmap System
- **Labels:** agentos
- **URL:** https://linear.app/jovie/issue/JOV-1932/build-roadmap-command-parser-for-linear-ops
- **Updated:** 2026-05-31T02:17:47.164Z

Implement the `/roadmap` command as a gstack skill or script wrapper around the Linear MCP. Required subcommands: `add` (create issue from brief), `expand` (generate sub-issues from an epic), `sync` (pull Linear state into `agentos/roadmap/`), `today` (emit today's active issues as agent-brief), `approved` (list issues with human-approval gate cleared), `agent-brief` (emit structured brief for a given issue ID). Each subcommand must be testable in isolation.
