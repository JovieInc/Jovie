The hook `.claude/hooks/conditional-decision-lint.sh` was created in this session but not yet wired into the agent harness. Tasks:

1. Decide which Claude Code lifecycle hook to bind it to (Stop, PostToolUse, or SessionEnd). Likely `Stop` (runs at task completion, same shape as existing `post-task-validate.sh`).
2. Add the binding to `.claude/settings.json` (or merge into `post-task-validate.sh` if folding is simpler).
3. Add a CI sanity test that pipes synthetic agent text through the hook and verifies:
   * Text with `follow-up later` and no JOV-ID and no EVENT: → fails in error-mode, warns in warn-mode
   * Text with `follow-up later JOV-1234` → passes
   * Text with `follow-up later EVENT: permanent taste rule` → passes
4. Hook starts in warn mode through 2026-06-01 (configured via `CONDITIONAL_DECISION_LINT_MODE` env, with auto-promotion to error after that date).
