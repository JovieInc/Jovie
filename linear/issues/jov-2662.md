# JOV-2662: UX: Normalize Library filters into top-level chips

- **Status:** Todo
- **Priority:** 🟠 High
- **Assignee:** Unassigned
- **Team:** JOV
- **Project:** —
- **Labels:** grok
- **URL:** https://linear.app/jovie/issue/JOV-2662/ux-normalize-library-filters-into-top-level-chips
- **Updated:** 2026-05-31T05:04:01.525Z

## Problem

Library filters are buried in a dropdown or sidebar. Users can't see what filters are active at a glance.

## Scope

* Move filter UI from dropdown to top-level chip/pill component
* Show active filters as colored chips below the header
* Add clear-all button
* Maintain existing filter logic, just change presentation

## Reference

JOV-7795 has existing context on this pattern

## Acceptance

* Filters visible as chips without opening dropdown
* Active state is visually clear
* Removing a chip clears that filter
