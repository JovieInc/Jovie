## Change

Replace the current phone-only hero image with a full-width desktop app screenshot showing the release system/dashboard, with a phone overlay on the right side showing the artist profile (fan-facing view).

## Why

* Phone-only hero = Linktree energy. We're a platform, not a link tool.
* Desktop screenshot shows the full stack: release system, audience data, dashboard — justifies the paid tier and signals "real product" to investors and artists alike.
* Phone overlay shows what fans see: clean profile, smart links, tips. Together they tell the whole story in one frame.

## Requirements

* Desktop screenshot should use **real data** (Tim White dashboard with actual releases) — no fake/mock data
* Phone overlay on the right side showing the public-facing artist profile
* Full-width layout on desktop, stacks gracefully on mobile
* Should feel like Linear/Notion/Vercel-tier hero presentation
