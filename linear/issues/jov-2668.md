## Problem

Global search (Cmd+K / Ctrl+K) has no keyboard navigation. Users can open it but can't navigate results without mouse.

## Scope

* Add arrow key navigation to search results
* Enter to select highlighted result
* Escape to close
* Focus visible on highlighted item (not just hover)
* Trap focus inside search modal when open

## References

* Reach UI MenuButton or Radix UI patterns
* Threads search for design reference

## Acceptance

* Arrow keys move through results
* Enter opens selected result
* Escape closes modal
* Focus is trapped inside modal
