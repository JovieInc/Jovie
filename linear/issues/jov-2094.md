## Problem

The admin Ops Kanban board is not using the full available width, and the card design does not follow current admin design principles.

## Required changes

1. **Full-width Kanban** — the board should span the full content area width. Columns should flex to fill the space rather than being constrained to a fixed or narrow container.
2. **Updated card design** — Kanban cards should follow these principles:
   * Compact: minimal vertical padding, tight typographic hierarchy
   * Surface: use `bg-surface-1` on the shell canvas, not nested cards-within-cards
   * No decorative borders or shadows beyond what the design system provides
   * One primary label, one secondary metadata line — no stacked chrome
3. **No status badges inside cards** (see companion issue) — status is already communicated by which column the card is in.
