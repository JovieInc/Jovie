# JOV-2665: UX: Refactor shell scrolling to keep right rail sticky

- **Status:** Todo
- **Priority:** 🟠 High
- **Assignee:** Unassigned
- **Team:** JOV
- **Project:** —
- **Labels:** grok

- **Updated:** 2026-05-31T07:23:15.000Z

## Problem

When scrolling the main content area, the right rail (sidebar) scrolls with the page instead of staying fixed. This makes the app feel broken.

## Scope

* Audit shell layout CSS (flex/grid structure)
* Make right rail position: sticky with proper top offset
* Ensure main content area scrolls independently
* Test at multiple viewport sizes

## Acceptance

* Right rail stays in place while main content scrolls
* No layout shift on scroll
* Works at 375px, 768px, 1440px
