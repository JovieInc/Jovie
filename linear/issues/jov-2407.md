# JOV-2407: Unify task toolbar search with shared header search action

- **Status:** Done
- **Priority:** 🟡 Medium
- **Assignee:** Tim White
- **Team:** JOV
- **Project:** Unified App Shell Migration
- **Labels:** design-system, area:app-router, area:ui, qa:e2e, polish, category:app-shell, parallelizable, ui, type:feature, ux
- **URL:** https://linear.app/jovie/issue/JOV-2407/unify-task-toolbar-search-with-shared-header-search-action
- **Updated:** 2026-05-31T03:48:49.347Z

## Slice

Align `/app/tasks` toolbar search with the shared `HeaderSearchAction` used by releases and admin tables.

## Scope

* Keep task filtering/data behavior unchanged.
* Replace the task toolbar's route-local desktop search input/clear button with the shared search action primitive.
* Preserve search/clear handlers, accessible labels, and compact toolbar layout.
* Add/adjust focused coverage for task search updates and clear behavior.

## Acceptance

* `/app/tasks` search still filters through existing task state.
* Toolbar search visual behavior matches the shared shell/header search primitive.
* No production imports from `app/exp/*`.
* Focused tests, typecheck, and desktop/mobile smoke pass.
