# JOV-1935: Build daily design proposal queue

- **Status:** Backlog
- **Priority:** 🟠 High
- **Assignee:** Unassigned
- **Team:** JOV
- **Project:** Design Lab
- **Labels:** agentos, area:infra
- **URL:** https://linear.app/jovie/issue/JOV-1935/build-daily-design-proposal-queue
- **Updated:** 2026-05-31T02:17:44.145Z

A cron generates N design proposals per day across registered design surfaces (see D4). Each proposal is persisted to `agentos/runs/design-lab/YYYY-MM-DD/` with surface ID, proposal text, generated asset references, and scoring metadata. Proposals are surfaced in the HUD at `/admin/ops` for human review. The cron must use the economy model tier; no premium calls without Tim approval.
